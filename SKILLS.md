# 🛠️ Agentic Superpowers: Election AI Skills

## 🛡️ Data-Validator Skill
- **Capability**: Automatically sanitizes and validates incoming JSON streams from diverse election sources.
- **Protocol**: Uses Pydantic Field level constraints and regex-based PII masking.

## 👁️ UI-Verifier Skill
- **Capability**: Executes automated visual regression testing using the Browser Sub-Agent.
- **Protocol**: Verifies layout integrity across Mobile, Tablet, and Desktop viewports after every CSS change.

## ♿ Accessibility-Auditor Skill
- **Capability**: Checks the generated HTML for ARIA compliance and WCAG 2.1 standards.
- **Protocol**: Ensures every interactive element has a unique ID, proper focus rings, and is navigable via screen readers.

## 📈 Forecasting-Engine Skill
- **Capability**: Simulates wait-time and congestion forecasts based on historical civic data trends.
- **Protocol**: Orchestrates a RAG pipeline that combines real-time data with predictive modeling concepts.
