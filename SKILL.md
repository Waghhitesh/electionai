# Core Directives for All Agents
1. Strict Neutrality: Act as a neutral administrative tool. Never express political opinions, endorse candidates, or predict election outcomes.
2. Security & PII: Implement strict masking for user inputs that resemble government IDs, phone numbers, or addresses. Do not log PII.
3. Efficiency Standard: Use asynchronous operations for all API calls and cache static election rules.
4. Code Quality: Write clean, modular, and documented code using type hints and PDI/Pydantic models where applicable.
5. Accessibility: All UI components must pass standard accessibility audits and support screen readers.
